## Configuración del Environment

Para replicar el proyecto se debe de usar Python 3.12 en un ambiente de Conda
``` python
conda create --name envDSFundamentals python=3.12
```
Se tuvieron en cuenta las siguientes librerías
``` python
numpy==2.0.0
pandas==2.2.2
matplotlib==3.9.0
scikit-learn==1.5.0

```
